  GOOGLE ТАБЛИЦА БЕЗ Google Cloud (если Cloud заблокирован)
═══════════════════════════════════════════════════════

Google Cloud требует 2FA — не нужен.
Используем Google Apps Script + обычный Google аккаунт.

───────────────────────────────────────────────────────
ШАГ 1. ТАБЛИЦА (2 мин)
───────────────────────────────────────────────────────
1. Chrome → https://sheets.google.com → пустая таблица
2. Назовите «Лиды Elena»
3. Скопируйте ID из URL:
   https://docs.google.com/spreadsheets/d/ ID /edit
4. Пришлите мне этот ID в чат

───────────────────────────────────────────────────────
ШАГ 2. APPS SCRIPT (5 мин)
───────────────────────────────────────────────────────
1. Chrome → https://script.google.com → Новый проект
2. Удалите код → вставьте весь файл leads-webhook.gs из проекта
3. В CONFIG вставьте только SHEET_ID (ID таблицы из шага 1)
   TG_BOT_TOKEN и TG_CHAT_ID оставьте пустыми — TG уже на Vercel
4. Сохранить (Ctrl+S)

5. Настроить таблицу (один раз):
   - Вверху выберите функцию: setupSheetLayout
   - Нажмите ▶ Выполнить
   - Разрешите доступ к таблице
   → Появятся колонки, SAPI, фильтры, подсветка дублей

6. Развернуть → Новое развёртывание:
   - Тип: Веб-приложение
   - Запуск от имени: Я
   - У кого есть доступ: Все
   - Развернуть

7. Скопируйте URL (заканчивается на /exec)
   Пришлите мне в чат

───────────────────────────────────────────────────────
ШАГ 3. Я ПОДКЛЮЧУ К ЛЕНДУ
───────────────────────────────────────────────────────
Добавлю URL в Vercel как GOOGLE_SHEET_WEBAPP_URL → Redeploy.

Лид с сайта → Vercel /api/leads → Telegram + строка в таблице.

───────────────────────────────────────────────────────
ЧТО ПРИСЛАТЬ МНЕ
───────────────────────────────────────────────────────
✅ ID таблицы
✅ URL веб-приложения (/exec)

Этого достаточно. JSON-ключи и Google Cloud не нужны.
