Вариант А (рекомендуется) — Google Sheets API
──────────────────────────────────────────────

Пришлите в чат **3 вещи**:

1. **ID таблицы** — из URL после создания пустой таблицы:
   https://docs.google.com/spreadsheets/d/ ЭТО_ID /edit

2. **Email сервисного аккаунта** — из JSON-ключа, поле client_email
   (выглядит как: elena-leads@project.iam.gserviceaccount.com)

3. **JSON-ключ целиком** — файл с Google Cloud (Service Account → Keys → JSON)
   ИЛИ только поле private_key из этого файла

Важно: таблицу нужно **расшарить** на email из п.2 с правом **Редактор**.

──────────────────────────────────────────────
Как получить JSON-ключ (5 минут, в Chrome):
──────────────────────────────────────────────
1. https://console.cloud.google.com/
2. APIs & Services → Library → «Google Sheets API» → Enable
3. Credentials → Create Credentials → Service Account
4. Keys → Add Key → JSON → скачать файл
5. sheets.google.com → новая таблица «Лиды Elena»
6. Поделиться → вставить client_email → Редактор

──────────────────────────────────────────────
Что будет в таблице (создаётся автоматически):
──────────────────────────────────────────────
24 колонки: Дата, SAPI, Имя, Email, Телефон, WhatsApp,
проблема, возраст, страна, проживание, зависть, CRM,
UTM-метки, fbclid/gclid/yclid/subid, страница.

SAPI — выпадающий список: новый | валид | квал | трэш | дубль
Дубли по телефону подсвечиваются автоматически.
